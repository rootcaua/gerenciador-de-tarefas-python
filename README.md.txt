📌 Gerenciador de Tarefas 

Um gerenciador de tarefas simples via terminal, desenvolvido em Python. Permite organizar atividades de forma prática, com controle de status e fácil visualização

🚀 Funcionalidades
✅ Adicionar tarefas
📋 Listar tarefas
🔄 Alterar status das tarefas
🔍 Filtrar tarefas por status
❌ Deletar tarefas
🧹 Limpar todas as tarefas
➕ Adicionar novos tipos de status


🛠️ Tecnologias utilizadas
Python 3
📂 Estrutura do projeto
  ─ main.py     
  ─ menu.py     

▶️ Como executar
Clone o repositório:
git clone https://github.com/seu-usuario/gerenciador-de-tarefas-cli.git

Acesse a pasta do projeto:
cd gerenciador-de-tarefas-cli

Execute o sistema:
python menu.py
